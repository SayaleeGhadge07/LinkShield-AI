# 🛡️ LinkShield AI

An AI-powered phishing URL detection system that analyzes suspicious URLs and provides intelligent risk assessments using Groq LLM.

---

## 🚀 Features

- Detects phishing URLs
- AI-generated explanations
- Risk Level Detection
- Modern responsive UI
- Built using n8n workflow automation

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- n8n
- Groq API

---

## Workflow

User enters URL

↓

Website sends URL to n8n Webhook

↓

Groq AI analyzes URL

↓

AI returns Risk Analysis

↓

Results displayed on Website
